# 🚀 Global Solution 2026 - Plataforma de Monitoramento Espacial

Projeto desenvolvido para a **Global Solution 2026** da disciplina de **Programação Orientada a Objetos (POO)**.

Uma estação espacial completa com monitoramento em tempo real, sensores, sistema de propulsão e gerenciamento de missão.

---

## 📋 Sobre o Projeto

Sistema de monitoramento espacial que demonstra os principais conceitos de POO:
- **Classe Abstrata**
- **Interface**
- **Encapsulamento**
- **Herança**
- **Polimorfismo**

---

## 🛠 Tecnologias Utilizadas

- **Java** (JDK 17 ou superior)
- Programação Orientada a Objetos
- Menu interativo via console
- Simulação de sensores com valores aleatórios

---

## ✨ Funcionalidades

- **✅ Sistema de Sensores** (Temperatura, Pressão e Radiação)
- **✅ Sistema de Propulsão** (Química e Elétrica)
- **✅ Dados da Missão** com encapsulamento e proteção por senha
- **✅ Sistema de Alertas** (ATENÇÃO, ALERTA e CRÍTICO)
- **✅ Menu interativo completo**
- **✅ Status em tempo real da estação**

---

## 🚀 Como Executar

1. Clone o repositório
2. Abra o projeto no **IntelliJ IDEA** (recomendado)
3. Execute a classe `SistemaMonitoramento.java`
4. Utilize o menu interativo
